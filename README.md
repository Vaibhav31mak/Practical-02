# 📌 Customer Account (.NET 10 Console App)

## 🚀 Overview
A console application that models a customer bank account using a class with constructor and method to display details.

## 🎯 Objective
- Understand classes and objects in C#
- Use parameterized constructors
- Practice method creation with documentation

## 🛠️ Tech Stack
- Language: C#
- Framework: .NET 10
- Type: Console Application

## 🧠 Concepts Covered
- Class & Object
- Encapsulation
- Constructor (Parameterized)
- Methods
- XML Documentation Comments

## ⚙️ How It Works
1. Create `Customer_Account` class with required fields
2. Initialize account number and customer name using constructor
3. Set bank name separately
4. Call `printInfo()` to display details

## ▶️ How to Run
1. Clone the repository
2. Navigate to project folder
3. Run:
   ```bash
   dotnet run
